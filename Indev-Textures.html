Indev Textures

A resource pack that bring back the old days.

Indev Textures is a resource pack for Minecraft Bedrock Edition and Java Edition that brings the old textures from Minecraft Java Beta and below, This resource pack will allow you to see the great old textures with loud sounds as old days

Features:

Adds back the old L O U D Sounds

Adds back the Pig without nose

Adds back old Enderman eyes (green eyes)

Adds back old Leather armor textures

Adds back Roses

Adds Neon Leaves and Grass

Adds back the epic d a r k w a t e r

Adds back Neon Sugarcane (going to be renamed to reeds in another version)

Adds back the old Stone logo

Bedrock edition Exlusive: Options to choose between (pre-1.14/1.14+ textures with beta and older sounds/new sounds with Indev textures of course):

Installation

Bedrock edition:

1. Download the pack and import it to Minecraft as usual.

2. (Recommended) Download Old Days UI Pack from here!

Recommended settings for more old Minecraft experience:

Brightness - 0%

Smooth Lighting - Off

Fancy Graphics - Off

UI Profile - Classic

Java Edition:

1. Download the pack and put it in resourcepacks folder in .minecraft as usual

2. (Recommended) Install Optifine

Recommended settings for more old Minecraft experience:

Brightness - Moody

Smooth Lighting - Off

(OPTIFINE EXCLUSIVE) Fog Start - 0.2

(OPTIFINE EXCLUSIVE) Sky - On

(OPTIFINE EXCLUSIVE) Sun & Moon - On

(OPTIFINE EXCLUSIVE) Dropped Items - Fast

(OPTIFINE EXCLUSIVE) Stars - On

(OPTIFINE EXCLUSIVE) Rain & Snow - On
