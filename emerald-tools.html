Emerald Tools

A add-on/mod that adds emerald tools to your game.

Have some emeralds but you don’t want to trade with them? Download this add-on to add these op tools to your game, and discard your diamond sword, diamond pickaxe, etc… now

Emerald tools add-on Add 4 new tools to the game.

Bedrock Edition:

I’ve been working on this Add-on for ~9 hours, made with bridge

You may want to check out the mod I made for Fabric Mod API for Java Edition

To get those tools you need to get them by “/give” command or crafting them

Crafting them is the same as other tools but with emeralds instead of diamonds, iron, gold, stone, and gold

Info and Crafting:

Every tool of these give you haste effect with amplifier 3

Emerald Sword:

The Emerald Sword do 4 attacks of damage.

Emerald Pickaxe:

The Emerald Pickaxe do 3 attacks of damage, sadly mines blocks by walking into them because Minecraft doesn’t allow making normal Pickaxe items.

Emerald Axe:

The Emerald Axe do 4 attacks of damage, sadly chops blocks by walking into them because Minecraft doesn’t allow making normal Axe items.

Emerald Shovel:

The Emerald Shovel do 2 attacks of damage, sadly breaks blocks by walking into them because Minecraft doesn’t allow making normal Shovel items.

Java edition:

Items:

Emerald Sword, Pickaxe, Axe, Shovel, and Hoe

Emerald Helmet, Chestplate, Leggings, and Boots

Emerald Apple

^ are worse than diamonds

Upgrades are available (Currently Iron and Quartz)

Installation

Bedrock edition:

1. Download the pack and import it to Minecraft as usual.

2. Go to your world settings.

3. in the BehaviorPacks menu add the Emerald Tools behavior pack.

4. in the ResourcePacks menu add the Emerald Tools resource pack.

Java Edition:

1. Download Fabric Mod loader.

2. Download this mod. (important for textures and items names)

3. Go to your mods folder, copy the jar file of the mod, paste it in the mods folder

4. ???

5. Profit
